# 🌸 Iris Flower Classifier

A Machine Learning web application that predicts Iris flower species using a Decision Tree Classifier.

## Features

- Predict Iris flower species
- Interactive Gradio web interface
- Built using Python and Scikit-learn
- Trained on the Iris dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Gradio
- Google Colab

## Project Structure

```
iris-flower-classifier/
│── Iris_Flower_Classification.ipynb
│── iris_model.pkl
│── requirements.txt
│── README.md
```
